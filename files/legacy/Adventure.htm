冒险模式命令指南

更改游戏模式

基岩版/Java 1.12及以下

/gamemode 2

/gamemode a

所有版本

/gamemode adventure

生成可放置的方块

基岩版

/give 玩家名 方块名 1 0 {"minecraft:can_place_on":{"blocks":["方块名",…,"方块名"]}}

Java 1.12及以下

/give 玩家名 方块名 1 0 {CanPlaceOn:["方块名",…,"方块名"]}

Java 1.13及以上

/give 玩家名 方块名{CanPlaceOn:["方块名",…,"方块名"]}

方括号内可以插入多个" 方块名"(包括英文引号)，之间用英文逗号(,)隔开。

生成方块破坏工具

基岩版

/give 玩家名 物品名 1 0 {"minecraft:can_destroy":{"blocks":["方块名",…,"方块名"]}}

Java 1.12及以下

/give 玩家名 物品名 1 0 {CanDestroy:["方块名",…,"方块名"]}

Java 1.13及以上

/give 玩家名 物品名{CanDestroy:["方块名",…,"方块名"]}

方括号内可以插入多个" 方块名"(包括英文引号)，之间用英文逗号(,)隔开。

示例：获得可破坏草方块、泥土和沙子的钻石锹

基岩版

/give @s diamond_shovel 1 0 {"can_destroy":{"blocks":["grass","dirt","sand"]}}

Java 1.12及以下

/give @s diamond_shovel 1 0 {CanDestroy:[grass,dirt,sand]}

Java 1.13及以上

/give @s diamond_shovel{CanDestroy:[grass_block,dirt,sand]}

示例：获得可放置在草方块和泥土上的石头

基岩版

/give @s stone 1 0 {"can_place_on":{"blocks":["grass","dirt"]}}

Java 1.12及以下

/give @s stone 1 0 {CanPlaceOn:[grass,dirt]}

Java 1.13及以上

/give @s stone{CanPlaceOn:[grass_block,dirt]}
